## ðŸš€ Getting Started

1. **Access Sanity Studio** at `/studio`
2. **Create New Content**:
   - Click "+" to add new documents
   - Choose between "Page" (flexible) or "Home Page" (specific)
3. **Add Sections**: Use the drag-and-drop interface to add and reorder sections
4. **Customize Content**: Each section has specific fields and options
5. **Preview Changes**: Use the visual editing features to see changes live
6. **Publish**: Set status to "published" when ready
